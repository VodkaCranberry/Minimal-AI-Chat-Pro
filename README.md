# Minimal AI Chat Pro

一个轻量级、纯前端的 AI Chat 网页客户端。

本项目使用原生 HTML、CSS 和 JavaScript 构建，无需后端服务即可运行。支持 OpenAI 兼容接口、多会话管理、历史记录搜索、文件与图片上传、Markdown 渲染、代码高亮、响应式布局、深色模式、本地配置保存等功能。

## 功能特性

- 纯前端实现，无需部署后端
- 支持 OpenAI 兼容 API 接口
- 支持自定义 API Base URL、API Key、模型名称、Temperature 和系统提示词
- 多会话管理
- 新建、删除、批量删除和清空会话
- 历史记录本地保存
- 支持导入和导出聊天记录
- 支持历史会话精准搜索
- 支持 Markdown 渲染
- 支持代码块高亮
- 支持代码一键复制
- 支持图片和文件上传
- 支持拖拽上传文件
- 支持直接粘贴图片或文件到输入框
- 支持流式输出
- 支持中断 AI 输出
- 支持重新编辑用户问题
- 每条消息支持复制为：
  - 纯文本
  - Markdown
  - DOCX/Word 友好的富文本格式
- 右侧对话进度节点，方便快速定位上下文
- 支持深色模式和浅色模式
- 支持手机、平板和桌面端响应式布局
